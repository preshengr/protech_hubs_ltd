# Protech Hubs Limited

## Project Overview

`Protech Hubs Limited` is a small business website built with static HTML, CSS, and image assets. It includes core company pages for presenting services, company information, a blog section, and a contact form page.

## Structure

- `index.html` - Home page with company introduction and featured content.
- `about.html` - About page describing the company, mission, and values.
- `services.html` - Services page listing offered solutions and expertise.
- `blog.html` - Blog page for news, updates, or industry articles.
- `contact.html` - Contact page with a form or contact details for inquiries.

## Asset Directory

- `asset/css/` - Contains the project stylesheet(s) used to style the website.
- `asset/img/` - Stores images used across the website, including logos, icons, and visuals.

## How to Use

1. Open the project folder in a code editor.
2. Serve the files locally or open `index.html` directly in a web browser.
3. Navigate to each HTML file to preview the website pages.

## Notes

- This is a static website and does not require a backend.
- The pages are designed to be easy to maintain and extend.
- Add new assets to `asset/img/` and update `asset/css/style.css` if you need new styles.

## Suggested Next Steps

- Add a navigation menu to connect all pages.
- Improve mobile responsiveness using CSS media queries.
- Add real content and images for the company branding.
- Enhance the contact page with a working form backend or third-party form service.
